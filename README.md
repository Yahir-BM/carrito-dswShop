Requiere de instalar en terminal:
- node express
- cors 
- facturapi

Lo mas probalble es que no lo deje correr de primera porque la falta del archivo
"carrito-key.json" ya que ese no se puede subir a git por las credenciales que tiene.

Ahora en Firebase cada usuario por id tiene su propio carrito de compras donde se van 
a poder manipular los productos disponibles.  
De momento en facturapi usa un tester_id por complicaciones de campos requeridos 
para que haga una factira.
